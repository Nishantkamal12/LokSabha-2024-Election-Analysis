# 🗳️ 2024 Lok Sabha Election Data Analysis using Python

A complete Python-based Data Analysis project built on the **2024 Indian General Election (Lok Sabha) Dataset**.

This project focuses on analyzing election results, vote share, party performance, constituency trends, and political insights using Python data analysis libraries and visualizations.

---

# 📌 Project Objective

The objective of this project is to:

- Analyze 2024 Lok Sabha Election Results
- Compare BJP, Congress, and Other Parties
- Generate meaningful political insights
- Create data visualizations using Python
- Practice real-world data analysis techniques

---

# 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

---

# 📂 Dataset

Dataset:
- 2024 Lok Sabha Election Results Dataset

The dataset contains:

- Candidate Name
- Party Name
- State
- Constituency
- Total Votes
- Winning Margin
- Vote Percentage
- Position

# 📌 Key Insights

- BJP won the highest number of seats in the 2024 elections.
- Congress improved its performance compared to previous elections.
- Regional parties remained influential in multiple states.
- Several constituencies had very close victory margins.
- Vote share and seat share patterns varied significantly.
